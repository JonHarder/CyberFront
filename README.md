# CyberFront

## build

### Dependencies

* Elm 0.19
* Node 8.11
* Yarn 1.9.4


### Development Server

```bash
yarn server
open localhost:8000
```

### Development Build

```bash
yarn build
```

### Production Build

```bash
yarn build-production
```
