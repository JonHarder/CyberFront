# CyberFront

## build

### Dependencies

* Elm 0.19
* Node 8.11
* Yarn 1.9.4


### Development Server

```bash
elm reactor
# assuming port 8000 was free
open localhost:8000/src/index.html
```

### Development Build

```bash
yarn build
```

### Production Build

```bash
yarn build-production
```
